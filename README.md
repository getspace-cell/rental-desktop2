# Rental Games Desktop

Настольное приложение для аренды игр на Windows 10/11.

## Описание

Приложение позволяет пользователям арендовать игры через ключи ПК клубов. После ввода ключа пользователь получает доступ к списку доступных игр и может запускать их через различные лаунчеры (Steam, Epic Games и т.д.).

## Возможности

- Ввод и безопасное хранение ключа ПК клуба
- Просмотр списка доступных игр
- Автоматический запуск игр через Steam
- Автоматический вход в аккаунты с поддержкой 2FA
- Блокировка доступа к Steam UI во время игры
- Мониторинг процесса игры
- Автоматическое завершение сессии при закрытии игры

## Требования

- Windows 10/11
- Python 3.8 или выше
- Установленные лаунчеры игр (Steam, Epic Games и т.д.)

## Установка

1. Установите Python 3.8+ с [python.org](https://www.python.org/downloads/)

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Использование

1. При первом запуске введите ключ ПК клуба
2. Перейдите в "Настройки" и укажите пути к лаунчерам игр
3. Выберите игру из списка и нажмите "Играть"
4. Приложение автоматически запустит игру и будет следить за процессом

## Структура проекта

- `main.py` - главный файл приложения
- `config.py` - управление конфигурацией и шифрование ключа
- `api_client.py` - клиент для работы с API бэкенда
- `steam_manager.py` - управление Steam процессами
- `game_launcher.py` - запуск игр
- `ui/` - интерфейс пользователя
  - `main_window.py` - главное окно
  - `key_input_dialog.py` - диалог ввода ключа
  - `settings_dialog.py` - диалог настроек

## Безопасность

Ключ ПК клуба сохраняется в зашифрованном виде в директории `%APPDATA%\RentalDesktop\`. Шифрование использует криптографически стойкий алгоритм на основе системной информации компьютера.

## API

Приложение работает с бэкендом по адресу: `https://passplay.ru`
